EcoPulse Media — учебный статический сайт

Состав архива:
- index.html — главная страница;
- services.html — страница услуг;
- event.html — страница мероприятия и пресс-релиза;
- style.css — оформление сайта;
- script.js — мобильное меню и демонстрационная форма;
- assets/ — логотип и авторские SVG-иллюстрации.

Как открыть:
1. Распакуйте архив.
2. Откройте файл index.html в браузере.
3. Для публикации загрузите папку на GitHub Pages, Netlify или любой хостинг статических сайтов.
